## Next js Quiz App

### How to run

#### 1. Clone Repository

```sh
git clone https://github.com/FadilRifqi/quiz.git
```

#### 2. Install Dependencies

```sh
npm install
```

#### 3. Build the Next Project

```sh
npm run build
```

#### 4. Run the Web Server

```sh
npm run start
```

#### Your website will run at http://localhost:3000 by default
